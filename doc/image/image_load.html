=====================================
8.2 Loading a .bmp image
=====================================
You can read data from a .bmp file (for example to draw an image using the background color of the console cells).

C++ : TCODImage::TCODImage(const char *filename)
C   : TCOD_image_t TCOD_image_load(const char *filename)

ParameterDescription
filenameName of the .bmp file to load.
Example :

C++ : TCODImage *pix = new TCODImage("mypic.bmp");
C   : TCOD_image_t pix = TCOD_image_load("mypic.bmp");
