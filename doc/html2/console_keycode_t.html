2.4.5. Key codes

TCOD_keycode_t is a libtcod specific code representing a key on the keyboard.
For python, replace TCODK by KEY: libtcod.KEY_NONE Possible values are :
When no key was pressed (see checkForKeypress) : TCOD_NONE
