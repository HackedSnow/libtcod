=====================================
16.1 Creating a generator
=====================================

 > 16.2 Generating a name
In order to be able to generate names, the name generator needs to be initialised with the proper data. It will then be ready to generate one specific type of names. The initialisation is done via:

C++ : TCODNamegen::TCODNamegen(const char * filename, const char * name, TCODRandom * random)
C   : TCOD_namegen_t * TCOD_namegen_new(const char * filename, const char * name, TCOD_random_t random)
Py  : ---

ParameterDescription
filenameThe file where the desired syllable set is saved, along with its relative parh, for instance, "data/names.txt".
nameThe name of the syllable set structure to use, for instance "fantasy female"
randomA random number generator object. Use NULL for the default random number generator
Note 1: The filename may contain various structures with syllable sets, but the generator will only use one such structure. Each file will be parsed once only, but the file name cannot be skipped even if a given file has been already parsed.
Note 2: The name refers to what stands between quotation marks after the declaration of a syllable set structure in the specified file. The name passed as parametre needs to exist in the specified file, otherwise an error will be triggered.

Example :

C++ : TCODNamegen myGenerator("data/names.txt","fantasy female",TCODRandom::getInstance());
C   : TCOD_namegen_t my_generator = TCOD_namegen_new("data/names.txt","fantasy female",TCOD_random_get_instance());
Py  : ---
